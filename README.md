# Jingqi Tong Personal Homepage

Static GitHub Pages site for `tongjingqi.github.io`.

## Files

- `index.html` - homepage content and publication links.
- `styles.css` - responsive academic homepage styling.
- `script.js` - active navigation highlighting.
- `.nojekyll` - tells GitHub Pages to publish the static files directly.

## Deploy

1. Create a public GitHub repository named `tongjingqi.github.io`.
2. Put these files at the repository root.
3. Commit and push to the `main` branch.
4. In GitHub, open `Settings > Pages` and choose `Deploy from a branch`, then select `main` and `/root`.
5. The site will be available at `https://tongjingqi.github.io/` after GitHub Pages finishes building.

## Edit Before Publishing

- Replace `Add current affiliation` with the current institution.
- Replace `Add public email address` with a public contact email or remove the email card.
- Add a `cv.pdf` link if a public CV should be shown.
- Review publication order and author lists against the final preferred CV format.
